# musicPlayer 
基于Vue的网页音乐播放器 [查看在线版本](https://kevinandrewdong.github.io/musicPlayer/)
- 使用网易云Node.js版API，实现歌曲地址，评论地址，mv地址的查询
- axios网络请求库，获取API数据
- v-on实现搜索按钮的绑定，v-bind设置歌src，v-for实现mvId非空时显示图标，v-show实现视频遮罩层
